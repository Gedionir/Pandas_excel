# Análise de dados com Python e Pandas
## Projeto 1 - Vendas por Região
* Natal se destaca com as maiores receitas
* 2019 apresentou maior volume de receita que 2018
* Identificação de padrões sazonais nas vendas mensais

## Projeto 2 - Adventure Works
* Cálculo de métricas financeiras (receita, custo, lucro)
* Análise de eficiência logística (tempo de envio)
* Identificação de outliers nos prazos de entrega
* Performance por marca e classe de produto

### Habilidades Demonstradas
* Leitura e concatenação de múltiplos arquivos
* Limpeza e tratamento de dados (valores nulos, tipos)
* Feature engineering (criação de novas colunas)
* Agrupamentos e análises segmentadas
* Filtros condicionais
* Visualizações diversas (barras, linhas, pizza, dispersão, histograma, boxplot)
